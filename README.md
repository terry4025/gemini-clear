# NanoStudio

> Gemini AI 워터마크 제거(NanoRemove)와 AI 기반 지능형 배경 제거/합성(NanoMatting)을 통합한 브라우저 기반 이미지 스튜디오

## ✨ 주요 기능

### 1. 🎯 워터마크 정리 (NanoRemove)
- **정밀한 수학적 복원** — Gemini AI 워터마크의 알파 블렌딩 공식을 역연산하여 픽셀 손실 없이 원본을 복원합니다.
- **자동 배치 처리** — 여러 이미지를 한 번에 드래그 앤 드롭하여 일괄 처리할 수 있습니다.
- **다양한 포맷 지원** — PNG, JPG, WebP 형식을 지원합니다.

### 2. ✂️ 배경 제거 및 합성 (NanoMatting) **New!**
- **온디바이스 AI 인스턴스** — Transformers.js(RMBG-1.4 모델)를 활용하여 브라우저 내에서 배경을 직접 분리합니다.
- **고품질 마스킹** — 로고의 색상을 훼손하지 않는 `destination-in` 기법과 엣지 보정 로직을 적용했습니다.
- **품질 미세 조정** — **'감도(Sensitivity)'** 슬라이더를 통해 배경의 안개 효과나 세밀한 경계면 보존 정도를 조절할 수 있습니다.
- **이미지 합성 스튜디오** — 투명 배경은 물론, 단색 배경 또는 사용자가 업로드한 이미지를 배경으로 즉시 합성할 수 있습니다.

## 🔒 개인정보 및 보안
- **100% 로컬 처리** — 모든 AI 연산과 이미지 처리는 사용자의 브라우저 내에서 수행됩니다. 이미지가 외부 서버로 전송되거나 저장되지 않아 프라이버시가 완벽히 보장됩니다.

## 🔬 기술 스택
- **Core**: Vanilla JavaScript (ES6+)
- **AI Engine**: @xenova/transformers (RMBG-1.4)
- **Styling**: Vanilla CSS (Modern Dark Aesthetic)
- **Icons**: Lucide Icons

## 🚀 시작하기

### 로컬 실행
`index.html`을 브라우저에서 직접 열거나 정적 서버를 통해 실행하세요:

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

### 사용 방법
1. 상단 탭에서 **워터마크 정리** 또는 **배경 제거 (AI)** 중 원하는 기능을 선택하세요.
2. 이미지를 업로드(클릭, 드래그, Ctrl+V)합니다.
3. 배경 제거 시, 상단의 **감도 슬라이더**로 품질을 조절하거나 **배경 이미지** 버튼으로 나만의 배경을 합성합니다.
4. 결과물을 다운로드하여 저장합니다.
